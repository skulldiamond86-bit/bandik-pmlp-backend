# Catatan Keuangan Harian - Backend

Backend REST API menggunakan Laravel 8 dan JWT Authentication.

## Teknologi

- Laravel 8
- PHP 8.1
- MySQL
- JWT Authentication
- REST API

## Fitur

- Login
- Register
- Logout
- CRUD Category
- CRUD Transaction
- Dashboard

## Cara Menjalankan

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
